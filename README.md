# Floors [BETA]

Floors is PHP based **login as-a service** platform for developing **single sign on** centralized or distributed web apps. 
visit [Floors](http://floors.cf) for see on working project results.

Main features:
* app management
* authorization
* users log
* roles

login features:
* Facebook
* Google Accounts
* Twitter
* Floors Account

... you can request
