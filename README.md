# Floors [BETA]

Floors is PHP based login as-a service platfrom for developing single sign on centalized or distributed web apps. visit [Floors](https://floors.cf) for try on working projects.

app main features:
* app management
* authorization
* users log
* roles

login features:
* facebook
* google accounts
* twitter (soon)
